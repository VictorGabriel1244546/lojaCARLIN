# Loja Arkkio
E-commerce de eletrodoméstico feito com:
* HTML
* CSS 
* JAVASCRIPT
* CARROSEL CSS

## Para instalar o projeto na sua máquina

```
$ git clone https://github.com/ErickBalbino/loja-arkkio.git
$ cd loja-arkkio
```
